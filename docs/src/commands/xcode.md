# Xcode

**Platforms:** macOS (`darwin`)

Switch between side-by-side Xcode installs and open Apple's download page.

## `switch-xcode-version`

Interactively point `/Applications/Xcode.app` at an `Xcode_*.app` install, run `xcode-select`, and accept the license. Aborts when `/Applications/Xcode.app` is a real directory instead of a symlink.

## `xcode-download-page`

Open the Apple Developer Xcode downloads page in the default browser.
